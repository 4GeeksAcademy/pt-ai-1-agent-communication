# Memory Bank Folder

This folder contains the persistent context files for the Night Sky Tracker application.

- `product-context.md`: Business and user context.
- `architecture.md`: Technical stack and design overview.
- `conventions.md`: Coding and style conventions that the agent should stick to.
- `decisions.md`: A living document containing any decisions made during the development process and why.
- `cadence.md`: Class schedule and time-boxed delivery rules.
- `plan.md`: Phased implementation plan and MVP milestones.
- `devlogs/`: This is where we will keep track of what we did in individual dev sessions.
